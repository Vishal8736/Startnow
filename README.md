# StartNow - Automated Pentesting Environment Setup

An autonomous script for Kali Linux to set up a complete Web and Network Pentesting environment.

## Features
- **Auto Root Login:** Automatically elevates privileges using preset credentials.
- **Virtual Environment:** Installs Python tools in a dedicated `venv`.
- **One-Command Access:** Use any tool by typing its name from any directory.
- **Tool Manager:** List all installed tools with `tools --list`.

## Tools Included
- **ProjectDiscovery:** Nuclei, Subfinder, Katana, Httpx, etc.
- **Tomnomnom:** Assetfinder, Waybackurls, Gf, etc.
- **Others:** FFuf, Arjun, Dirsearch, Gf-Patterns, and Vishal8736's tools.
